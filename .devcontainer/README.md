# Cokra Dev Container

This directory contains the configuration for developing Cokra in a Dev Container.

## Usage

1. Install [Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) extension
2. Open this project in VS Code
3. Press F1 and select "Dev Containers: Reopen in Container"

## Features

- Rust toolchain
- Bazel build system
- All required dependencies pre-installed
